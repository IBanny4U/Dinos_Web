ကြာဇံချက်

ပါဝင်ပစ္စည်းများ
     • ကြက်သား - (၅၀) ကျပ်သား၊
     • ပဲကြာဇံ - (၁၅) ကျပ်သား၊ 
     • မှိုခြောက် ပန်းခြောက် - (၅) ကျပ်သား၊
     • အသားတု - (၅) ကျပ်သား၊ 
     • ငုံးဥ - အလုံး (၂၀)၊
     • ကြက်ဥ - (၅) လုံး၊
     •ကြက်သွန်နီဥ - (၁၀) လုံး၊
     • နံနံပင်အနည်းငယ်၊ 
     •ရှောက်သီး -(၁) ခြမ်း 
    ပြုလုပ်နည်း

ကြက်သားမကိုင်ခင် ပဲကြာဇံနဲ့ မှိုခြောက် ပန်းခြောက်၊ အသားတုတို့်ကို ဇလုံတစ်လုံးစီမှာ သပ်သပ်စီထည့်ပြီး ရေစိမ်ထားပေးလိုက်ပါ။ ငုံးဥနဲ့ ကြက်ဥကိုလည်း ရေနွေးအိုးထဲ ထည့်ပြုတ်ထားပေးလိုက်ပါ။ ပြီးရင် ကြက်သား (၅၀) ကျပ်သားကို ရေပြောင်အောင်ဆေးပြီး အတုံးသေးသေးလေးတွေ တုံးပေးရပါမယ်။ ကြက်သားကို အတုံးတုံးပြီးရင်တော့ ဘေးခဏဖယ်ထားလိုက်ပါ။ ဒယ်အိုးတစ်လုံးထဲမှာ ဆီထည့်ပြီး အပူပေးလိုက်ပါ။ ပြီးရင် ငရုတ်သီးမှုန့် (အရောင်တင်မှုန့်) နဲ့ ကြက်သားကိုထည့်ပြီး ဆီသတ်ပေးလိုက်ပါ။ မိနစ်အနည်းငယ်ကြာအောင် ဆီသတ်ပြီးရင်တော့ ကြက်သားအိုးကို မီးပိတ်ပြီး မီးဖိုပေါ်က ချနိုင်ပါပြီ။ ဇောက်နက်တဲ့ အိုးတစ်လုံးထဲမှာ ရေ (၃) ပုံ (၂) ပုံခန့်ထည့်ပြီး ရေနွေးအိုးတည်ပေးလိုက်ပါ။ ပြီးရင် ခုနက ကြက်သားဆီသတ်ထားတာကို ရေနွေးအိုးထဲ ထည့်ပေးလိုက်ပါ။ ပြီးရင် ခဏလောက် မွှေပေးလိုက်ပါ။ ကြက်သွန်နီဥ ၇လုံးလောက် အိုးထဲ အခွံနွှာပြီး ထည့်ပေးပါ။ ကြက်သားအိုး ပွက်ပွက်ဆူလာပြီဆိုရင် ရေစိမ်ထားတဲ့ ပဲကြာဇံဖတ်တွေနဲ့ မှိုခြောက် ပန်းခြောက် အသားတုတွေကိုရေထဲကဆယ်ပြီး အိုးထဲထည့်ပေးလိုက်ပါ။ ပြုတ်ထားတဲ့ ငုံးဥတွေကို အခွံခွာပြီး ကြာဇံချက်အိုးထဲ ထည့်ပေးလိုက်ပါ။ ကြက်ဥကိုတော့ အခွံခွာ၊ လေးခြမ်းခြမ်းပြီး ကြာဇံချက်အိုးထဲ ထည့်ပေးလိုက်ပါ။ ပြီးရင် ငရုတ်ကောင်းမှုန့်လေး နည်းနည်း ဖြူးပေးလိုက်ပါဦး။ ပြီးရင်တော့ မီးပျော့ပျော့နဲ့ ခဏလောက် တည်ထားပေးလိုက်ပါ။ ပြီးရင်တော့ ပန်းကန်လုံးတစ်လုံးထဲ ခပ်ထည့်လိုက်ပြီး အပေါ်ကနေ ကြက်သွန်နီနဲ့ နံနံပင်ကို ပါးပါးလှီးပြီး ဖြူးပေးလိုက်ပါ။ ရှောက်သီးစိတ်လေးတွေကိုတော့ ကြိုက်တတ်ရင် ညှစ်စားလို့ရအောင် ဘေးမှာ ရံပေးထားလိုက်ပေါ့။ ဒါဆိုရင် အရသာရှိတဲ့ ကြာဇံချက်လေးကို လုပ်စားလို့ရပါပြီနော်။

ပြင်ဆင်ပြုလုပ်နည်း
၁။ ကြက်သားအကောင်လိုက်ကို သန့်စင်ပြီး ခပ်သေးသေးတုံးထားပါ။အသားချည်းပဲကြက်သားများကိုလဲ အတုံးအသေးလေးတွေတုံးထားပါ။ ကြက်သွန်ဖြူကို ထောင်းထားပြီး တဝက်စီခွဲထားပါ။ ဂျင်းကိုလဲထောင်းထားပါ။
၂။ ဇလုံတခုထဲကို ရေနွေး ၁ ခွက်၊ ရေအေး ၃ ခွက်ထည့်ပြီး ပဲမှုန့်ထည့်၍ ပဲမှုန့်ပျော်သွားအောင်ဖျော်ပါ။ ပဲမှုန့်အလုံးလေးများဖြစ်နေလျှင် လက်ဖြင့် ချေပေးပါ။
၃။ အိုးကြီးကြီး (ခန့်မှန်းချေ ၇လီတာခွဲဆံ့သော) တခုထဲမှာ ရေ ၅ လီတာ ထည့်ရျ် ပွက်လာအောင်တည်ပါ။ ပွက်လာလျှင် ဆနွင်းလက်ဖက်ရည်ဇွန်း တဝက်၊ ဆား လက်ဖက်ရည်ဇွန်း ၄ဇွန်း၊ သကြား ထမင်းစားဇွန်း ၃ ဇွန်း၊ ကြက်သားမှုန့်ထမင်းစားဇွန်း ၁ ဇွန်းတို့ကို ထည့်ပါ။ ထို့နောက် ဖျော်ထားသော ပဲမှုန့်များကို ဇကာခံ၍ ထည့်ပါ။
၄။ ဒယ်အိုးတခုထဲမှာ ဆီ ထမင်းစားဇွန်း ၈ ဇွန်း ကိုထည့်၍ အပူပေးပါ။ ပူလာလျှင်ထောင်းထားသော ကြက်သွန်ဖြူ တဝက်၊ထောင်းထားသောဂျင်းတို့ကို ဆီသတ်ပါ။မွှေးလာလျှင် ဆနွင်း လက်ဖက်ရည်ဇွန်း တဝက်ကို ထည့်ဆီသတ်ပြီးမှ အရောင်တင်မှုန့်လက်ဖက်ရည်ဇွန်း ၁ ဇွန်းခွဲကိုထည့်ဆီသတ်ပါ။ အရောင်တင်မှုန့်ထည့်ပြီးပြီးချင်း ကြက်အကောင်လိုက်မှ တုံးထားသော ကြက်သားများကိုထည့်၊ ယိုးဒယားငံပြာရည် ထမင်းစားဇွန်း ၃ ဇွန်းထည့်၊ သကြား (သို့) အချိုမှုန့်လက်ဖက်ရည်ဇွန်း ၁ ဇွန်းတို့ကိုထည့်ရျ် လုံးပါ။ (အရောင်တင်မှုန့်ထည့်ပြီး နည်းနည်းမွှေပြီး ကြက်သားများချက်ချင်းထည့်ပါ။ နောက်မို့အရောင်တင်မှုန့်များတူးသွားတတ်ပါတယ်။)
၅။ ထို့နောက်လုံးထားသော ကြက်သားများကို ပဲရည်အိုးထဲထည့်ပါ။ ပဲရည်အိုးကို ခဏခဏမွှေပေးရပါတယ်။ ကြက်သားထည့်ပြီး နောက်တခါထပ်ပွက်လာလျှင် ကြက်သွန်ဖြူ ဆီချက်နှင့် ကြက်သွန်နီဆီချက်ကိုထည့်ပါ။(ကြက်သွန်ဖြူနီဆီချက်ကိုထည့်လျှင်ပိုမွှေးပါတယ်။ အလွယ်တကူမရှိလျှင်မထည့်လဲရပါတယ်။) ပြီးလျှင် အုန်းနို့ကိုထည့်ပါ။ (အုန်းနို့ကိုထည့်တာစောလျှင် လဲဟင်းရည်အိုးသိုးတတ်ပါတယ်။) ထို့နောက်မီးအေးအေးဖြင့် တည်ထားပါ။ တည်ထားတုံးမှာ ရေကိုထပ်ထပ်ဖြည့်ပေးပါ။ ( မီးအေးအေးနဲ့ ၃ နာရီလောက်ထားပါတယ်။) ဟင်းရည်ထဲကို ကြက်သွန်နီဥများကြိုက်နှစ်သက်ပါကထည့်ပါ။
၆။ နောက်ထပ် ဒယ်အိုးတခုထဲမှာ ဆီ ထမင်းစားဇွန်း ၂ ဇွန်းကိုထည့်၍ အပူပေးပါ။ ဆီပူလာလျှင် ကျန်သော ကြက်သွန်ဖြူတဝက်ကိုထည့်၍ ဆီသတ်ပါ။ မွှေးလာလျှင် ဆနွင်းလက်ဖက်ရည်ဇွန်း တဝက်၊ အရောင်တင်မှုန့်လက်ဖက်ရည်ဇွန်း ၁ ဇွန်းကိုဆီသတ်ပါ။ တုံးထားသောအသားချည်းကြက်သားများကိုထည့်ပါ။ ရေအနည်းငယ်ထည့် တည်ပါ။ ရေခြောက်သွားလျှင် မီးဖိုပေါ်မှချပါ။
၇။ အုန်းနို့ဟင်းရည်ထဲမှကြက်သားများနူးလာလျှင် မီးဖိုပေါ်မှချပါ။
၈။ အုန်းနို့ခေါက်ဆွဲပြင်ရန်အတွက် ဂျုံခေါက်ဆွဲကို ပန်းကန်ထဲအရင်ထည့်၊ ဟင်းရည်ကိုအပေါ်ကလောင်း၊ အပေါ်ကနေ ချက်ထားသော ကြက်သားထည့်၊ ပဲကြော်၊ ကြက်ဥပြုတ်၊ အကြွပ်ကြော်၊ ပါးပါးလှီးထားသောကြက်သွန်နီတို့ကိုထည့်၍ပြင်ပါ။ သုံးဆောင်နိုင်ပါပြီရှင်။

ရေအများကြီး မထည့်မိစေရန် သတိထားပါ

မုန့်သားစစ်ရန် ဇလုံတစ်ခုကို အောက်ခံထားပြီး ဇလုံပေါ်မှာ အပေါက် ခပ်စိတ်စိတ်ပါသော ဇကာကို တင်ပေးပါ

ပေါင်းအိုးအောက်ခံ အိုးထဲကို ရေတစ်ဝက်သာသာ ဖြည့်ပေးပြီး ဆူအောင်တည်ပေးပါ

        ဆား
        မြေပဲစံ
        နှမ်းလှော်
ပြုလုပ်နည်း 
သရက်သီးတစ်လုံးကို ဂျစ်လေးနဲ့ ခြစ်ပြီး ဆားနဲ့နယ်ပြီးဆေးပါတယ်ရှင်  ပြီးရင် ဆီချက်ချက်ပါတယ်။
        မြေပဲစံ နှမ်းလှော်လေးလည်းထောင်းထားပါတယ်။
        သရက်သီး ‌ဆေးထားတာကို ရေသေချာညစ် ဆီချက်ထည့်ပါမယ် ကြက်သွန်နီ ပါးပါးလှီးထားတာထည့်ပါမယ် ငပိကတော့ ထောင်းထားပြီးသားဘူးလေးထည့်ပါတယ် ပဲမှုန့် မြေပဲထောင်း အရသာ ထည့်မယ် ငရုတ်သီးမှုန့် / ငရုတ်သီးစိမ်းလေးထည့်ပါမယ် 
        ကဲ သေချာနယ်ပြီး စားနိုင်ပါပြီ
        ထမင်းပူပူလေးနဲ့ အရမ်းလိုက်ဖက်ပြီး စားကောင်းသော သရက်သီးသုပ်လေးပါရှင်

သခွါးသီးရခိုင်သုပ်

ရခိုင်တွေရဲ့ ပင်တိုင်နည်းအတိုင်း ငရုတ်သီးနဲ့ ငပိကို သုံးပြီး သုပ်မှာပါ။

      ပြင်ဆင်ပြုလုပ်နည်း
      - သခွါးသီးကို ရေဆေးပြီး နှစ်ခြမ်းခြမ်းပေးပါ။ အစေ့နဲ့ အူတွေကို ဖယ်ပေးပါ။

      - သခွါးသီးအသားကို အမျှင်သိပ်မထူစေဘဲ သေချာခြစ်ပေးပါ။ 
- ကြက်သွန်နီ ပါးပါးလှီးပေးထားပါ။ ငရုတ်သီး ၊ ငပိ မီးကင်ပြီး ထောင်းပေးပါ။ 

      - ခြစ်ထားတဲ့ သခွါးသီး ၊ လှီးထားတဲ့ ကြက်သွန်နီနဲ့ ထောင်းထားတဲ့ ငရုတ်သီးငပိထောင်း ကို 
          ရောနယ်ပေးပါ အပေါ့ အငံ မိမိကြိုက်သလိုထည့်ပေးပါ။  
  
      - သံပုရာသီး ညှစ်ပေးပါ။ နောက်ဆုံးမှာ အဖြူရောင် နံနံပင် ပင်ပေါက်လေးကို ထည့်လေ့ရှိပါတယ်။ 
        
      - ချဉ် ငံ စပ် အရသာ နဲ့ သခွါးသီးရခိုင်သုပ်လေးကို ရပါပြီ။

        ငါးပိ၊ ကြက်သွန်ဖြူ နှင့် ငရုတ်သီးစိမ်းတို့ကို မီးဖုတ်ပြီး ထောင်းပါ။ 
        ကြက်သွန်နီ နှင့် နံနံပင်ကို လှီးထားပါ။ ငရုတ်ကောင်းကို ထောင်းပါ။ 
        အပေါ်ကလုပ်ထားသော အစာများအကုန်ကို ဆား၊ အချိုမှုန့်တို့နှင့်ရော၍ သံပုရာရည်ညှစ်ပြီး သုပ်ပါ။ 
        ငါးမန်းသားခြောက်သုပ်ရပါပြီ။

ခရမ်းသီး ငသရွဲခြောက်မီးဖုတ်သုပ်

  ( မြန်မာ အခေါ် - ခရမ်းသီး ငါးတံခွန် အခြောက် မီးဖုတ်သုပ်)

ပြင်ဆင်ပြုလုပ်နည်း
      - ရခိုင်လို အင်ဗာရီ လို့ခေါ်တဲ့ ခရမ်းသီး အကြီးကြီး ကို စို့လေးနဲ့ တစ်လုံးလုံး 
        နေရာအနှံ့ထိုးပေးပါ။ ပြီးရင်တော့ မီးသွေးမီးဖိုထဲ ထည့်ပြီး ဟိုဘက် ဒီဘက်
        ပြန်လှန် ဖုတ်လိုက်ပါ။
- ငါးတံခွန် အကောင်သေးကို သေချာ ပွတ်တိုက်ပေးပြီး ကြွပ်လာအောင်ထိ 
        မီးဖုတ် / ကင်ပေးပါ။ 

      - မိမိ နှစ်သက်သလောက် ငရုတ်သီးစိမ်း နဲ့ ငပိစိမ်းစားလေး နည်းနည်း မီးကင်ပေးထားပါ။

      - ခရမ်းသီး မွှေးပြီး အိလာပြီဆိုရင် ထုတ်ပြီး ရေနဲ့ ပွတ်ဆေးပေးလိုက်ပါ။ အပေါ်အရေခွံ
        ခွါပေးပါ။ ပြီးရင် ဇလုံတစ်ခုထဲထည့်ပြီး ချေပေးပါ။

      - ငါးတံခွန် ကြွပ်လာပြီဆိုရင် ရေနဲ့ပွတ်ဆေးပေးပြီး (ရေသွန်ပြီး မဆေးပါနဲ့၊ အီသွားမှာ စိုးလို့ပါ) 
        သေးသေးလေးတွေ ပိုင်းပြီး ၊ ခရမ်းသီးဇလုံထဲ ထည့်ပေးပါ။

      - မီးကင်ထားတဲ့ ငရုတ်သီး နဲ့ ငပိကို ဂျင်းအနည်းငယ် ထည့်ထောင်းပေးပါ။ 
        (ဂျင်းမထည့်ဘဲလည်း စားကြပါတယ်) 

      - ကြိုက်တတ်ရင် ကြက်သွန်ဖြူ ထည့်ထောင်းပေးပါ။ (ထည့်လေ့ထည့်ထတော့ မရှိသလောက်ပါပဲ) 

      - ကြက်သွန်နီ ပါးပါးလှီးထားပေးပါ။ ပြီးရင်တော့ ခရမ်းသီးနဲ့ ငါးခြောက် ထည့်ထားတဲ့ 
        ဇလုံထဲကို ငရုတ်သီးထောင်း ၊ ကြက်သွန်နီပါးပါးလှီးထား တာ ထည့်ပေးပြီး 
        အချို အငန် မိမိသဘောကျထည့်ပြီး ရောနယ်ပေးပါ။

      - မကျည်းနှစ်လေး ရေဖျော်ထည့်ပေးပါ။ 

      - နောက်ဆုံး နံနံပင်အုပ်ပေးလိုက် ရင် မီးဖုတ်နံ့ သင်းသင်းလေးနဲ့ မွှေးအိနေတဲ့ ခရမ်းသီး ငါးတံခွန်သုပ်ကို ရပါပြီ။

        ငါးနီတူ ကိုကျွတ်အောင်လှော်အအေးခံ, (ကြိုတင်လှော်ပီး အေးမှလေလုံပုလင်းနဲ့သိမ်းထားလဲရသည်)

        ကြက်သွန်နီပါးပါးလှီး ရယ်
        ရှောက်ရွက်ရယ်ကလွဲ ကျန်တာ အကုန်ရောသုပ်
        သမပီဆိုမှ ကြက်သွန်နီပါးပါးလှီး
        အချို အငံမြည်းပြီးရင် သံပုရာသီး ညှစ်ထည့်ပါ။
        နောက်ဆုံးမှ နံနံပင်လေး အုပ် ပေးလိုက်ပါ။ 
        ရှောက်ရွက် ခပ်ဖွဖွရောနယ်ပီး အပေါ့အငံမြည်းပီးသုံးဆောင်လို့ရပါပြီ။
        ငါးနီတူကို ရခိုင်ကျောက်ကျောနှင့်လဲရောစပ်သုပ်သည်။
        ပါဝင်ပစ္စည်းကတော့အတူတူပါ။
        မှတ်ချက် - ရခိုင်သုပ်တွင် ဆီချက်၊ ပဲမှုန့်လုံးဝမပါဝင်ပါ။
